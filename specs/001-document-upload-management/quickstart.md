# Quickstart: Document Upload and Management

## Prerequisites
- .NET 8 SDK
- Windows OS (for demo)
- No production credentials required

## Setup
1. Clone the repository (if not already)
2. Open solution in Visual Studio or VS Code
3. Build the solution: `dotnet build`
4. Run the app: `dotnet run --project ContosoDashboard/ContosoDashboard.csproj`

## Usage
1. Log in with any demo user (mock authentication)
2. Navigate to the Documents section
3. Upload files (PDF, Office, images, text; ≤25MB)
4. Add required metadata (title, category, project)
5. View, filter, and search documents
6. Download, preview, and share documents
7. Edit metadata, replace files, or delete documents
8. Upload from task page or dashboard widget
9. Receive notifications for shares/uploads

## Notes
- All features work offline/local only
- No real virus scanning or production security
- All actions are logged for audit (demo only)
- See README for limitations and training focus
